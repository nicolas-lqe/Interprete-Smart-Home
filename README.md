### 🧠 Descripción  
Este trabajo práctico consiste en el diseño e implementación de un intérprete para un lenguaje orientado a eventos llamado **SMART-HOME**, enfocado en la automatización de hogares.  

El sistema incluye:  
- Análisis léxico (Lexer)  
- Análisis sintáctico (Parser)  
- Validación de scripts  
- Traducción automática a HTML
